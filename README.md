# Praktisk Dag (PWA)

En enkel kalender-app til praktiske dage med:
- dagside med tekst og billeder
- automatisk gemning i browseren
- PWA-understøttelse (kan installeres på telefon)

## Kør lokalt

Åbn projektet via en lokal webserver (ikke `file://`).

## Struktur

- `html/index.html` – kalenderoversigt
- `html/day.html` – side for en valgt dag
- `css/styles.css` – styling
- `js/app.js` – app-logik
- `manifest.webmanifest` + `sw.js` – PWA
